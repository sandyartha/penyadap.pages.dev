## 5. Txtract MMS & SMS

- - -

Txtract adalah aplikasi Android yang dimana fungsinya dapat dengan mudah mengambil data komunikasi dari HP seseorang dengan sangat mudah ditambah ada fitur salin ke email, dimana kita dapat menyalin pesan target ke email kita setiap kali ada SMS masuk atau keluar di hp target. 

**UPDATE!** Sekarang **txtract** dapat menyadap / mengambil pesan melalui pc (**komputer**) jadi agan dapat melihat semua data SMS melalui pc atau juga laptop dimana pun. 

Hasil sadap dari aplikasi Txtract dapat kita pindah kan ke dalam file .**CVS** .**XML** .**HTML** agar lebih mudah kita melihat penyadapan data dari hp seseorang. 

Sebelum pasangan jangan lupa kalau aplikasi ini di pasang di hp target, *bukan di hp mu ya gan* HeHehe. Juga jangan sampai ketahuan oleh si doi kalau hp dia sedang diawasi.

**Menginstall**

1. Pertama download terlebih dahulu aplikasi > **[Txtract MMS & SMS.](https://play.google.com/store/apps/details?id=com.brilaps.android.txtractpro.withmms&hl=en)**
2. Setelah pemasangan selesai maka langkah selanjutnya. 
3. Setelah ada tampilan pada aplikasi pilih > **Extract All.**
4. Setelah di extract semua data yang ada hp target maka langkah selanjuntya adalah mengirim data ke email > **Email the output** > Pilih > **OK.**
5. Pilihan untuk ekstraktor dengan konversi file **TXT**, file **XML** ditampilan pilihan lainnya **CVS** atau **HTML** tinggal dipilih saja mau yang mana agan akan di ekstrak. 

![SADAP HP ANDROID TXTRACT](/wp-content/2022/02/17/sadap-hp-android-txtract-compressor.jpg)

**Download Txtract MMS & SMS**

## 7. SMS Backup & Restore

- - -

Aplikasi yang ketiga **SMS Backup & Restore** dengan cara sadap menggunakan aplikasi ini sangat bagus untuk digunakan buat agan.

Karena Anda dapat mengambil data komunikasi salah satu contoh seperti daftar panggilan telepon, yang bisa agan ambil dari hp target melalui data gmail.

**Fitur**

* Sadap / Backup SMS / MMS (pesan teks) pesan dan / atau log panggilan dalam format XML.
* Mengambil data dengan pilihan untuk secara otomatis upload ke Email,  **Google Drive** atau **Dropbox**.
* Sadap / Backup MMS (Pesan Gambar)
* Sadap **Pesan Teks** dan **Log Panggilan** ke ponsel lain melalui *Wi-Fi Direct*.
* XML backup dapat di konversi ke format lain, dan juga dapat dilihat di komputer.

**Menginstall SMS Backup & Restore**

**1.** Download terlebih dahulu aplikasi >> **[SMS Backup & Restore](https://play.google.com/store/apps/details?id=com.riteshsahu.SMSBackupRestore)** 

**2.** Tunggu hingga download selesai 

**3.** Apabila download telah selesai kemudian buka aplikasinya setelah kamu buka maka ada beberapa tampilan menu, selanjutnya.

**4.** Klik “**Restore**“tombol, file backup dalam format xml akan tercantum dalam antarmuka. Jika anda ingin memeriksa data pada backup, dapat memilih salah satu yang anda inginkan dan tekan “**Browse**” tombol di sudut kanan bawah.

Namun, untuk mengaktifkan fungsi ini, anda perlu menginstall sebuah aplikasi file manager seperti **ES File Explorer** atau **File Manager OI**.

![sadap sms hp android](/wp-content/2022/02/17/sadap-sms-hp-android-3-1.jpg)

**5.** Kemudian, pilih opsi “**Restore All Messages**” atau “**Only Restore Messages Send / Received Since**“**:** 

![SADAP SMS HP ANDROID](/wp-content/2022/02/17/sadap-sms-hp-android-3-21.jpg)

**6.**  Setelah selesai, anda dapat memeriksa jumlah total SMS. 

![SADAP HP ANDROID PANGGILAN TELPON](/wp-content/2022/02/17/sadap-sms-hp-android-3-31.jpg)

**7.** Pilih “**View**” pilihan dalam antarmuka utama. Kemudian, centang satu cadangan file dan tekan “**OK**“ tombol, akan melihat percakapan di dalamnya. 

Tekan satu untuk lihat full konten, termasuk waktu penerima dan pengirim. Jika menemukan salah satu yang ingin untuk mengambil di Android, dengan tekan yang lama sampai muncul “Percakapan Opsi” kotak muncul.

Klik “**Restore Percakapan ini**” tombol untuk memulihkan SMS yang dipilih. 

![CARA SADAP HP ANDROID](/wp-content/2022/02/17/sadap-hp-android-4-1.jpg)

**8.** Selain hanya mengambil data **SMS untuk HP Android** atau tablet, anda juga dapat mengirim  hasil sadap ke mail atau Gmail untuk digunakan lebih lanjut. 

![SADAP HP ANDROID EMAIL](/wp-content/2022/02/17/cara-sadap-hp-android-5.jpg)
