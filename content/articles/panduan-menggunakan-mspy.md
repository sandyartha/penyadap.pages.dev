# Panduan Lengkap Menggunakan mSpy untuk Pemantauan

mSpy adalah aplikasi pemantauan yang powerful untuk melindungi keluarga Anda. Berikut adalah panduan lengkap menggunakannya.

## Fitur Utama mSpy

### 1. Pemantauan Lokasi
- Tracking lokasi real-time
- Riwayat lokasi yang dikunjungi
- Geofencing untuk area aman

### 2. Pemantauan Media Sosial
- WhatsApp
- Instagram
- Facebook Messenger
- Telegram
- Snapchat

### 3. Kontrol Aplikasi
- Lihat aplikasi terinstall
- Blokir aplikasi berbahaya
- Monitor penggunaan aplikasi

### 4. Pemantauan Panggilan & SMS
- Riwayat panggilan
- Pesan teks (SMS)
- Kontak tersimpan

## Cara Menggunakan Dashboard mSpy

1. Login ke dashboard mSpy
2. Pilih perangkat yang ingin dipantau
3. Akses berbagai fitur pemantauan
4. Atur notifikasi untuk aktivitas mencurigakan
5. Ekspor data untuk dokumentasi

## Tips Keamanan

- Pastikan penggunaan legal dan berizin
- Jaga kerahasiaan akses dashboard
- Backup data secara berkala
- Update aplikasi secara rutin
